# Template React-Three-fiber project
## Commands
1. Git ignore
Command clear cache : 
```bash
git rm -r --cached .
```

2. GLTFJSX
Create tsx model
```bash
npx gltfjsx model.glb -t
```

3. Clone
Create tsx model
```bash
git clone https://github.com/YaelBusser/r3f-react-hooks.git name
cd name
del .git
del .idea
git init
git remote add origin githublink
```
